# Bundesliga Quiz Generator

## Overview
The Bundesliga Quiz Generator is an AI-powered application designed to create engaging football quizzes and personalized learning experiences for users interested in the Bundesliga. This project leverages current football data to generate quizzes tailored to user preferences, enhancing their knowledge and enjoyment of the sport.

## Features
- **AI-Powered Quiz Generation**: Utilizes advanced algorithms to create quizzes based on user interests and current football statistics.
- **Personalized Learning Paths**: Adapts to user performance, providing customized learning experiences to improve knowledge retention.
- **Interactive Quizzes**: Engaging football trivia questions with multiple-choice answers.
- **Progress Tracking**: Monitors user performance over time, allowing for adjustments in learning paths.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd bundesliga-quiz-generator
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage
To start the application, run:
```
npm start
```
Follow the on-screen instructions to generate quizzes and track your learning progress.
